Notes Application

Минималистичный текстовый редактор на базе Python и PySide6 с интегрированным модулем текстового анализа.
Приложение представляет собой доступный инструмент для создания и редактирования текстовых документов. Программа ориентирована на чистый интерфейс, исключающий лишние элементы управления.

Основные возможности:

    Управление файлами: создание, открытие и сохранение документов в кодировке UTF-8.

    Контроль изменений: интеллектуальная система отслеживания правок с визуальной индикацией в заголовке окна и защитой от случайного закрытия несохраненных данных.

    Персонализация текста: динамическая настройка шрифта и размера кегля через встроенные компоненты управления.

    Статистика: мониторинг количества символов в режиме реального времени.

    Review AI (в будущем Shorokh AI): специализированный модуль для глубокого анализа текста и получения рекомендаций по его улучшению.

Технологический стек:

    Язык программирования: Python 3.x

    Графическая библиотека: PySide6 (Qt for Python)

    Дизайн интерфейса: Qt Designer (UI-файлы конвертированы в Python-модули)
